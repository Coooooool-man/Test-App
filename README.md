Здравствуйте. Приложение запускается с класс MainApp (src/main/java/org/example/testapp/ui/MainApp.java). 
Тесты в папке Test.

В resourses - макеты приложения (форма таблицы и форма создания/редактирования) и файл стилей.

Приложение использует слоистую архитектуру. В папке model - модель пользователя User. В паке dao - класс и интерфейс для взаимодействия с БД. 
В папке util - класс обертка с одним методом для подлючения к бд. Настройки для кодключения в файле .env (он сам загружается).
В папке ui - классы графического интерфейса (MainController - форма таблица пользователей, UserDialog - форма создания и редактирования).
